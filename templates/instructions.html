Data Import App

Data Import App for Different Hospitals

Instructions

This Data Import App handles bulk import of data based upon easy to use CSV template with the following conditions:

CSV FORMAT

Required Columns
  1. Table : Patient Information
    • Registration Number
    • Gender
    • DOB or Age
    • Location Type
    • Country
    • District
    • State
  2. Table : Hospital Patient Relation
    • Registration Number
    • Hospital Department
    • Date of Admission
    • Location Type
  3. Table : Sample Information
    • Lab Sample ID
    • Sample Type
    • Sample Collection Date
  4. Table : Susceptibility Testing
    • Organism
    • Identification Method
    • Value
    • Validation status
